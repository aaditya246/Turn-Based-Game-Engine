# Turn-Based-Game-Engine
Turn-Based Game Engine | C++



• Engineered a turn-based game engine in C++, strictly adhering to SOLID and OOP principles to create a
highly modular and maintainable codebase.


• Decoupled game state, logic, and AI modules using the Single Responsibility Principle, reducing code
complexity and simplifying future updates.


• Architected an extensible system using the Open/Closed Principle, allowing for the seamless integration of
new AI opponents with zero changes to existing code.
